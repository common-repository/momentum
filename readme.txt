Momentum is a simple plugin to repost the home page with new content appended daily. Google sitemaps are submitted daily.

== Description ==

Momentum allows you improve search engine ranking with a few tricks. 
= Improved Search Engine Ranking =
Everyday Momentum will download a news snipit and save it locally for fast access
Any page that has the shortcode will be updated everyday automatically
The Sitemap will be submitted to Google and Bing everyday

= Simple instructions =
After installing and activating the plugin
Add the short code [addm2p url=""] in your home page were you want it and update the page.
View the page to check you are happy!!!

= Who needs this Plugin =
Every Website owner
A lot of websites are just for business presence and are very rarely changed. Search engines have complex algorythms but they are focused on new content.
By doing a daily page update will help a page do better in Search engine listings.

= Required Plugins =
This plugin assumes you have a plugin too keep your sitemap.xml file uptodate.


== Installation ==

1. Upload the `momentum` folder to the `/wp-content/plugins/` directory
2. Activate the momentum plugin through the 'Plugins' menu in WordPress
3. copy the ShortCode [addm2p url=""] into the homepage, normally at the bottom.
